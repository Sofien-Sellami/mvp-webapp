Yet another Vue app...

## Setup

1. Clone the Repo 

```bash
git clone https://github.com/openroboticmetaverse/mvp_frontend.git

```

2. Docker Compose

```bash
docker exec -it mvp_frontend /bin/bash
```
```bash
docker exec -it mvp_frontend /bin/bash
```
3. In the container console

```bash
cd mvp_frontend
```
```bash
npm i
```
```bash
npm run dev
```

Enjoy!
